## 🚀 **Quick Start (5 minutes)**

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the Dashboard
```bash
python run_dashboard.py
```

### 3. Open Browser
- Go to: `http://localhost:8501`
- The dashboard will automatically train models and load data
